Why no plugins?

Plugins add extra complexity and add a security risk. Therefore plugin-free solutions are easier and cheaper to host. If you use no plugins, you can simply choose to host your Jekyll project on GitHub Pages. GitHub Pages allows you to deploy your site for free on GitHub, with a custom domain name!
